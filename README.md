# LILLY‑i (Intelligence) Platform — Functional Prototype

This is a **clickable, local, no-backend** prototype dashboard for an AI-powered intelligence layer for **LILLY Direct**:

- **Geographic Heat Map** (where to expand)
- **Disease Prioritization Matrix** (what conditions to prioritize)
- **Competitive Radar** (where competitors are gaining share)
- **Regulatory & Policy Watchtower** (what policy changes matter)
- **“So What” Command Center** (synthesized actions + recommendations)

## Run

1. Open `index.html` in your browser (Chrome/Edge recommended).
2. No install required (charts use CDN libraries).

## Notes

- Data is **synthetic demo data** generated client-side in `data.js`.
- “AI” insights are a deterministic **scoring + anomaly detection + rules engine** in `insights.js` (meant to be swapped later for real models + real feeds).
